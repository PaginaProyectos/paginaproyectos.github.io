Descripción

Este aplicativo web, sirve para el manejo y control de expedientes para vista al publico y de personal es decir lleva un control de la persona anclada a ese expediente con el organismo que lleve el caso del mismo tambien cuenta con sistema de roles de administradores y usuarios cada quien con sus vistas y paneles ademas de contar seguridad de autenticacion la cual permite que ningun usuario pueda ver lo de otro usuarios a menos que se le permita.
